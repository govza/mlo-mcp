# mlo-mcp

## Agent skills

### Issue tracker

Hybrid: official issues live in GitHub Issues (`govza/mlo-mcp`, via `gh`); work-in-progress issues and specs live as gitignored markdown under `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

The five canonical labels, used as-is: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` at the repo root is the glossary and entry point into the reference docs under `docs/`; new decisions go to `docs/adr/`. See `docs/agents/domain.md`.
